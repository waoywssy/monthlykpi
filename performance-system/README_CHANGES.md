## 更新说明

新增了以下功能：

1. **数据库结构更新**：
   - 添加了 `Project` 模型，包含名称、客户、总金额、状态等信息
   - 添加了 `Collection` 模型，关联到项目，记录回款金额、日期、期间等信息
   - 设置了级联删除关系

2. **后端 API 接口**：
   - `/api/projects`: GET 获取项目列表，POST 新建项目
   - `/api/projects/[id]`: PUT 更新项目，DELETE 删除项目
   - `/api/collections`: GET 获取回款列表（支持按项目和期间筛选）
   - `/api/collections/upload`: POST 解析并导入 Excel 数据

3. **前端页面**：
   - `/projects`: 项目管理页面，实现增删改查
   - `/collections`: 项目回款页面，展示回款列表，支持按项目和期间筛选，展示数据汇总，支持 Excel 导入功能

4. **导航栏更新**：
   - 添加了"项目管理"和"项目回款"两个菜单项

所有的代码变更都已构建通过。导入功能可以解析包含"项目名称"、"客户名称"、"回款金额"、"回款日期"等列的 Excel 表格（兼容"2026年市场回款统计表"的格式）。
